说明

  • 9200 用于外部通讯，基于http协议，程序与es的通信使用9200端口。
  • 9300 jar之间就是通过tcp协议通信，遵循tcp协议，es集群中的节点之间也通过9300端口进行通信。
  • -p 端口映射
  • -e discovery.type=single-node 单点模式启动
  • -e ES_JAVA_OPTS=”-Xms128m-Xmx512m”：设置启动占用的内存范围
  • -v 目录挂载
  • -d 后台运行
